# Student Result Management System (Python)

This is a Python-based project that allows adding, viewing, searching, updating, and deleting student details using JSON file storage.

## Features
- Add student details  
- View all students  
- Search by roll number  
- Update marks and grade  
- Delete student records  
- Data stored using JSON file
